Инструкция по установке, настройке и использованию электронной формы учебника

 

Настоящая инструкция пользователя описывает работу с основными компонентами приложения — средствами просмотра учебника.

 

Технические требования

 

Операционная система:

— Windows 8 и выше.

Платформы:

— Apple iOS;

—  Android.

 

Устройства:

— стационарный компьютер (в том числе с подключением к интерактивной доске);

— ноутбук (нетбук);

— планшетный компьютер;

— смартфон.

 

Возможности электронной формы учебника

 

Структура, содержание и художественное оформление электронной формы учебника соответствует печатной форме издания.

 

·       Учебник имеет возможность:

— создания активных рукописных заметок, создания индивидуальных закладок;

— перехода на нужную страницу и обратно;

— мгновенного поиска слов.

 

 — инструкция.

 

Инструменты

 

I. Верхняя панель

 «Поиск» предназначен для поиска слов в учебнике. В поле ввода следует ввести текст для поиска и нажать «Поиск». Пользователь увидит список всех случаев, когда выбранное слово встречается в данном учебнике. Щёлкнув по какому-либо из найденных примеров, пользователь перейдёт на соответствующую страницу, где найденное слово будет выделено жёлтым цветом.

 

II.                 Боковая панель

     «Правая стрелка» перемещает на следующую страницу электронного учебника.

     «Левая стрелка» перемещает на предыдущую страницу электронного учебника.

 

III.             Нижняя панель

  «Масштабирование» позволяет менять масштаб отображаемой на экране страницы. При нажатии данной кнопки появляется навигация:. Здесь находятся кнопки дальнейшего увеличения, уменьшения, перехода на предыдущую, следующую страницы, изменения способа передвижения.

 «Миниатюры страниц» позволяют просматривать полосу страниц учебника в уменьшённом виде.  Щёлкая на разных участках полосы прокрутки, пользователь может быстро переходить на соответствующие страницы учебника.

 «Навигация страниц» делает перемещение к странице, следующей за текущей страницей, возвращает пользователя на страницу, которая была открыта перед последним действием, открывает доступ к первой, последней страницам учебника, показывает номер текущей страницы и общее количество страниц в учебнике.

 «Закладка» позволяет создавать закладки для быстрого перехода на нужную страницу.

 «Заметки» позволяют добавлять свои записи, просматривать уже существующие заметки. Созданную заметку можно в дальнейшем удалить или изменить.
